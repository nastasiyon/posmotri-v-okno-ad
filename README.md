# Посмотри в окно

Одностраничное приложение для просмотра видео из окон разных городов России. Пользователь может фильтровать видео по названию города и времени суток, переключаться между роликами и подгружать дополнительные результаты.

## Функциональность

### Загрузка данных
При открытии страницы автоматически загружается первая порция видео (5 штук) из API. На время загрузки поверх видео и списка карточек отображается прелоадер.

### Фильтрация
Форма поиска позволяет фильтровать видео:
- **Город** — текстовый поиск по названию (без учёта регистра)
- **Время суток** — чекбоксы «Утро», «День», «Ночь» (можно выбрать несколько)

При отправке формы список и видео обновляются. Если ничего не найдено — отображается сообщение об ошибке.

### Переключение видео
Клик по карточке в списке загружает соответствующее видео в основной плеер. Активная карточка подсвечивается.

### Пагинация
Если результатов больше 5, в конце списка появляется кнопка «Показать ещё». По клику догружается следующая страница карточек, список расширяется.

## API

Данные загружаются с `https://v-content.practicum-team.ru/api/videos`.

Параметры запроса:

| Параметр | Описание |
|---|---|
| `pagination[pageSize]` | Количество результатов на странице |
| `pagination[page]` | Номер страницы |
| `filters[city][$containsi]` | Фильтр по городу (без учёта регистра) |
| `filters[time_of_day][$eqi]` | Фильтр по времени суток (`morning`, `afternoon`, `night`) |

## Запуск

Достаточно открыть `index.html` в браузере. Для корректной работы шрифтов и запросов к API рекомендуется запускать через локальный сервер:

```bash
# Например, через расширение Live Server в VS Code
# или через npx:
npx serve .
```

## Состояния интерфейса

| Состояние | Описание |
|---|---|
| Initial | Страница загружена, показан прелоадер, затем первые 5 видео |
| Результаты найдены | Список карточек, активное видео, кнопка «Показать ещё» (если есть) |
| Нет результатов | Сообщение «Нет подходящих видео =( » поверх видеоблока |
| Ошибка сети | Сообщение «Ошибка получения данных :(» |
